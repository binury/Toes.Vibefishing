## Known issues

- Updating pixelization value in settings will not reload the shader with your new settings. (Until you restart the game)
  This will be added in a later update, soon!

- A few models/species (green fur only?) look like they are sprouting 5 o'clock shadow 😹

- Mod config (colors) are disregarded and changing them has no effect.
  This will also be enabled in a later update, very soon!

- Rainfall effect interpolation/transition time is altered regardless of whether pixelization is disabled if mod is enabled.
